- CONTROLS:

	* M: Create marine
	* N: Create enemy marine
	* G: Create ghost
		* Keypad 1: Invisibility
	* F: Create firebat
	* Spacebar: return cam initial position
	* F1 : Debug mode
	* P: pause the game
